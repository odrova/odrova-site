# ODROVA — кастомизация темы InSales

Файлы для черновика **«ODROVA new»** в InSales (demo-smart в фирстиле).
Рабочая тема #628976 НЕ трогается.

GitHub = архив/история. InSales = вставка вручную через редактор шаблона.

## Цель
Привести InSales к виду GitHub-прототипа odrova.ru: тёмный фон, фирменный
логотип oDROVA.RU, жёлтый+графит, прямоугольные кнопки посадки.

## Фирменный стиль
- Жёлтый #f5c800, графит #1e1e1e, чёрный #111
- Шрифты: Bebas Neue (лого/цены), Russo One (заголовки), PT Sans (текст)
- Угловатые срезы кнопок (clip-path)

## Как применять
В InSales: Редактировать шаблон → раздел **Стили** → файл `custom.scss`
→ вставить ВСЁ содержимое assets/custom.scss в конец.

Логотип работает через CSS (псевдоэлементы), правка header.liquid НЕ требуется.
