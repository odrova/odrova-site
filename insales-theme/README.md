## Уроки/грабли (чтобы не наступать снова)
1. **@import шрифтов работает только ДО первого CSS-правила** — держать первой строкой.
2. MutationObserver, который пишет в наблюдаемое поддерево, зацикливается —
   писать только при реальном изменении значения (setText с проверкой).
3. Тема перебивает кастомные правила: поднимать специфичность префиксами
   (`.od-product-page .od-product__info ...`), а старые конфликтующие блоки УДАЛЯТЬ.
4. Страница товара светлая через `body:has(.od-product-page)`; шапке при этом
   нужно явно вернуть тёмный фон (она прозрачная).
5. Коды виджетов могут жить в настройках Метрики («HTML-разметка для всех страниц»),
   а не в файлах темы — искать и там.
6. Переименовали свойство варианта в админке → обновить ключ в Products.setConfig
   (product.liquid), иначе кнопки превратятся в выпадашку. Ключ — посимвольно.
7. Перед Ctrl+V в редакторе темы смотреть на АКТИВНУЮ ВКЛАДКУ, а не на список
   слева. CSS в шаблоне/описании товара = «простыня» текста на странице.
